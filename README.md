# 💰 Financio

**Financio** is a personal finance management system designed to help users track their income, expenses, account transfers, and currency conversions in an organized way.

This project serves as a complete demonstration of the following skills:

- Relational database modeling with **MySQL**
- Software development using **Python** with **Object-Oriented Programming (OOP)**
- **Data analysis** and **visualization** using Python libraries
- Application of **Machine Learning** techniques for financial prediction and classification
- Clean and professional project structure and documentation

---

## 🚧 Project Status

> 🔧 Currently in the **Database Design Phase**

---

## 🎯 Project Goals

- Track income, expenses, and transfers across accounts
- Support multiple currencies and store historical exchange rates
- Categorize transactions with flexible category/subcategory management
- Generate visual reports and insights over time
- (Optional) Use Machine Learning to predict future expenses or detect anomalies
