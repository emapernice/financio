# Financio

**Financio** is a personal finance management system designed to help users track income, expenses, transfers, and currency exchanges across multiple accounts.

## 🧭 Project Objectives

- Showcase relational modeling and MySQL best practices.
- Implement core backend logic using Python and Object-Oriented Programming (OOP).
- Include data analysis capabilities with optional extensions into machine learning.
- Build a modular and extensible codebase.

## 🧱 Database Design

The database schema includes support for:

- Users and multiple accounts
- Income, expenses, and transfers
- Categories and subcategories
- Supplier and currency tracking
- Currency exchange with historical rates

The full schema is available in the file: [`schema.sql`](./schema.sql)

## 🛠️ Tech Stack

- **Backend:** Python 3.x (OOP)
- **Database:** MySQL
- **ORM:** SQLAlchemy (optional)
- **Frontend:** Optional / TBD
- **Visualization / Analysis:** pandas, matplotlib, scikit-learn (for ML phase)
